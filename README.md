# Lazarus简明使用手册

## 简介
Lazarus是一个免费的开源集成开发环境（IDE），专为使用Free Pascal编译器开发Pascal语言程序而设计。Lazarus由Pascal编写而成，其界面风格与Delphi相似，提供了强大的程序开发功能。无论你是Pascal语言的初学者还是经验丰富的开发者，Lazarus都能为你提供一个高效、便捷的开发环境。
